WELCOME TO 小钉装饰工程设计有限公司

小钉装饰工程设计有限公司是一家集设计，施工，软装饰于一体的盐城本地的专业装饰公司，专注于家居空间，并承揽办公，商业，娱乐空间，酒店宾馆写字楼等装饰工程的设计与施工，公司成立时间虽然不长，但在各重要岗位的员工从事装饰行业的年限最短的都已经超过五年的时间，尤其在设计方面，我们拥有一支眼光独到，经验丰富，自信尽职的专业团队，在施工管理方面，我们采用第三方监理的管理模式,由宜居丁渤监理公司，专门监督管理我公司的施工工艺及施工质量。
